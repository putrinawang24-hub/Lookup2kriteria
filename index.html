Advanced Excel: LOOKUP dengan 2 Kriteria

Cara mencari data berdasarkan dua nilai sekaligus.

Seringkali, satu kriteria saja tidak cukup. Anda mungkin perlu mencari "Gaji" untuk karyawan bernama "Budi" di departemen "Sales", karena mungkin ada "Budi" lain di departemen "IT".

Kita akan gunakan contoh tabel data ini:

Kolom A
(Nama Depan)		 Kolom B
(Departemen)		 Kolom C
(Gaji)
Andi Sales 5.000.000
Budi IT 7.000.000
Andi IT 8.000.000
Budi Sales 6.000.000

Tantangan: Bagaimana cara kita mencari Gaji untuk "Budi" di departemen "Sales"? (Jawabannya harus 6.000.000).

Metode 1: Kolom Bantu (Cara Klasik Termudah)

Metode ini bekerja di semua versi Excel dan paling mudah dipahami. Kita membuat satu "kunci unik" buatan.

Langkah-langkah:

  1. Buat Kolom Bantu: Tambahkan kolom baru di sebelah kiri data Anda (misal: di Kolom A baru).
  2. Gabungkan Kriteria: Di kolom baru (sel A2), gabungkan Nama Depan (B2) dan Departemen (C2). 
    =B2 & C2

    Tarik formula ini ke bawah. Anda akan mendapatkan "AndiSales", "BudiIT", "AndiIT", "BudiSales".

  3. Gunakan VLOOKUP: Sekarang lakukan VLOOKUP biasa menggunakan "kunci unik" baru Anda. 
    =VLOOKUP("BudiSales", A:D, 4, FALSE)

    (Mencari "BudiSales" di kolom A, mengembalikan data dari kolom ke-4, yaitu Gaji).

Kelebihan:

Sangat mudah dimengerti dan 100% kompatibel dengan semua versi Excel.

Kekurangan:

Menambah "sampah" di data Anda (kolom bantu) dan tidak elegan.

Metode 2: XLOOKUP (Cara Modern Terbaik)

Jika Anda menggunakan Excel 365/2021+, ini adalah cara terbaik. Bersih, cepat, dan tidak perlu kolom bantu.

Sintaks:

Kita menggunakan logika boolean (TRUE/FALSE). Kita mencari nilai 1 (yang artinya TRUE).

=XLOOKUP(1, (range_kriteria1 = kriteria1) * (range_kriteria2 = kriteria2), range_hasil)

Contoh:

=XLOOKUP(1, (A2:A5="Budi") * (B2:B5="Sales"), C2:C5)
Cara Kerjanya:
  1. (A2:A5="Budi") menghasilkan: {FALSE; TRUE; FALSE; TRUE}
  2. (B2:B5="Sales") menghasilkan: {TRUE; FALSE; FALSE; TRUE}
  3. Saat dikalikan (*), Excel memperlakukan TRUE=1 dan FALSE=0.
    {0;1;0;1} * {1;0;0;1} = {0; 0; 0; 1}
  4. XLOOKUP mencari angka 1 di array hasil {0;0;0;1}, menemukannya di posisi ke-4.
  5. Ia lalu mengembalikan nilai di posisi ke-4 dari range_hasil (C2:C5), yaitu 6.000.000.

Kelebihan:

Sangat bersih (tanpa kolom bantu) dan logikanya jelas.

Kekurangan:

Hanya berfungsi di Excel 365 / 2021+.

Metode 3: INDEX + MATCH (Cara Klasik Profesional)

Ini adalah cara para "pro" melakukannya sebelum XLOOKUP ada. Logikanya mirip XLOOKUP, tapi sintaksnya lebih rumit.

Sintaks:

Ini adalah "Array Formula". Di Excel 2019 ke bawah, Anda harus menekan Ctrl + Shift + Enter setelah mengetik formula ini.

{=INDEX(range_hasil, MATCH(1, (range_kriteria1 = kriteria1) * (range_kriteria2 = kriteria2), 0))}

Contoh:

=INDEX(C2:C5, MATCH(1, (A2:A5="Budi") * (B2:B5="Sales"), 0))

(Jangan lupa tekan Ctrl + Shift + Enter jika Anda tidak menggunakan Excel 365).

Cara Kerjanya:
  • Logikanya identik dengan XLOOKUP.
  • MATCH(1, ..., 0) mencari posisi angka 1 dari hasil perkalian array. (Ia akan menemukan posisi ke-4).
  • INDEX(C2:C5, 4) kemudian mengambil nilai ke-4 dari kolom Gaji (C2:C5).

Kelebihan:

Sangat kuat dan berfungsi di hampir semua versi Excel (Excel 2007+).

Kekurangan:

Sintaks rumit dan butuh Ctrl+Shift+Enter di versi lama (mudah lupa).

Rekomendasi Ahli

1. Pilihan Terbaik: XLOOKUP

Jika Anda pakai Excel 365/2021+, gunakan ini. Ini adalah standar baru.

2. Pilihan Termudah: Kolom Bantu

Jika Anda butuh kompatibilitas total atau ingin cara yang paling mudah diajarkan ke tim.

3. Pilihan Kompatibel (Pro): INDEX+MATCH

Jika Anda butuh kompatibilitas di versi lama (2007-2019) tapi tidak mau menambah kolom bantu.
